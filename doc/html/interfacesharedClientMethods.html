Dokumentacja funkcji składowych

void sharedClientMethods.checkIfOverwritingFiles ( String  position,
String  option 
) throws jabelko

Sprawdza, czy wybrana opcja download/upload może spowodować nadpisanie wybranego pliku po drugiej stronie połączenia.

Parametry
position- numeryczny identyfikator pliku z danej listy.
option- literał określający, którą listę należy przejrzeć.
Wyjątki
jabelko- wyjątek zawiera wartość boolean isOverwrite, która przyjmuje wartość TRUE, kiedy w wyniku wybranej operacji zostanie nadpisany wybrany plik. Przyjmuje wartość FALSE w przeciwnym wypadku.
